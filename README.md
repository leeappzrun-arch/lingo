## Run

With [uv](https://docs.astral.sh/uv/) (recommended):

```sh
uv run python -m lingo
```

Or with a plain venv:

```sh
python -m venv .venv
source .venv/bin/activate
pip install -e .
python -m lingo
```

## How to play

- Type letters `A`‚Äď`Z` to fill the current row.
- `ENTER` submits the guess.
- `BACKSPACE` deletes the last letter.
- After each guess, letters are colored:
  - **green** ‚ÄĒ correct letter, correct spot
  - **yellow** ‚ÄĒ letter is in the word, wrong spot
  - **grey** ‚ÄĒ letter is not in the word
- `ESC` returns to the main menu.
- `q` quits from the menu, `Ctrl+C` quits from anywhere.
